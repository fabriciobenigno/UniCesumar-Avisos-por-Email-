Configuração da mensagem

Exemplo: aluno1@email.com, aluno2@email.com ou um por linha.
A URL de envio já está configurada. Você pode alterar depois, se quiser, mas o sistema já fica pronto para uso.
Como usar:
1. Escreva o assunto e a mensagem.
2. Cole os e-mails dos alunos.
3. Visualize à direita.
4. Para envio real em massa, basta manter a URL do Google Apps Script configurada.
5. O layout sempre sairá com a logo unicesumar no topo e o botão botaosuporte no final.
